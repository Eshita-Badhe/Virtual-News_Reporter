Virtual News Reporter is an AI-based voice-controlled system that delivers personalized, real-time news summaries. Built with HTML, CSS, JS, Python, Gemini AI, GNewsAPI, and Google TTS/STT, it lets users select categories, listen to updates, and interact with the AI, making news more engaging and accessible.

<br>

<img src="https://github.com/Eshita-Badhe/Virtual_News_Reporter/blob/main/images/Fig1.jpg" alt="Virtual News Architecture" style="width:50%;height:auto;">

## Features
<ul>
    <li><strong>Voice-Based Interaction:</strong> Users can speak to navigate, choose categories, and request summaries.</li>
    <li><strong>Real-Time News:</strong> Fetches the latest news using GNewsAPI.</li>
    <li><strong>Summarized Content:</strong> AI-generated concise summaries for quick understanding.</li>
    <li><strong>Interactive AI:</strong> Users can discuss topics or ask for more details using natural language.</li>
    <li><strong>Text-to-Speech & Speech Recognition:</strong> Google TTS & STT integration for smooth hands-free use.</li>
    <li><strong>Personalized Feed:</strong> Tailored content based on user-selected categories.</li>
</ul>

## Technologies Used
<ul>
    <li><strong>Frontend:</strong> HTML, CSS, JavaScript</li>
    <li><strong>Backend:</strong> Python</li>
    <li><strong>APIs:</strong> GNewsAPI</li>
    <li><strong>AI:</strong> Gemini AI</li>
    <li><strong>Voice Processing:</strong> Google Text-to-Speech (TTS) & Speech-to-Text (STT)</li>
</ul>

## Installation

### Prerequisites
<ul>
    <li>Python installed on your system</li>
    <li>Basic terminal/command line knowledge</li>
</ul>

### Steps
<ol>
    <li>Clone the repository:
        <pre><code> git clone https://github.com/Eshita-Badhe/Virtual-News_Reporter.git</code></pre>
     </li> 
  <li>Navigate to the project directory:
        <pre><code>  cd Virtual-News_Reporter</code></pre>
    </li>
    <li>Install dependencies:
        <pre><code>pip install -r requirements.txt</code></pre>
    </li>
    <li>Run the app:
        <pre><code>python app.py</code></pre>
    </li>
</ol>

## Usage
<ol>
    <li>Open the application in your browser.</li>
    <li>Use voice commands to select a news category.</li>
    <li>Listen to summarized reports.</li>
    <li>Ask follow-up questions or switch topics hands-free.</li>
</ol>
